MurkaUserBot Installation

Simple, guided installation with Heroku:

bash -c '. <('"$(command -v curl >/dev/null && echo 'curl -Ls' || echo 'wget -qO-')"' https://raw.githubusercontent.com/MurkaUserBot/MurkaUserBot/master/install.sh) '"$*"


Default installation:

git clone https://github.com/MurkaUserBot
cd MurkaUserBot
python3 -m friendly-telegram
